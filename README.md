# 💻 Projeto ASP.NET MVC - Curso FullStack SENAC Osasco

Este é um projeto desenvolvido durante o curso de **FullStack do SENAC Osasco**, com o objetivo de aplicar na prática os conceitos de desenvolvimento web utilizando **ASP.NET MVC**.

---

## 🚀 Tecnologias utilizadas

- ASP.NET MVC
- C#
- .NET Framework / .NET Core
- HTML5
- CSS3
- Entity Framework (se estiver usando)

---

## 📚 Sobre o projeto

O projeto é um sistema de atendimento para uma clinica médica e  foi criado com fins educacionais para praticar:

- Estrutura MVC (Model-View-Controller)
- Criação de páginas dinâmicas
- Integração com banco de dados
- Boas práticas de desenvolvimento web
- Organização de código
